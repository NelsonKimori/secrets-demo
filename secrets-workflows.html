Secrets & Workflows

This case study highlights secure handling of secrets within CI/CD pipelines. Instead of hardcoding sensitive values, I integrated secure secret management.

Key Highlights

  • Used GitHub Actions Secrets to store API keys and credentials
  • Configured workflows to inject secrets securely at runtime
  • Prevented secrets from being exposed in logs or commit history
  • Added automated checks to detect accidental secret leaks
